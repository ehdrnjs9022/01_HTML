[용어 정리]
        Hyper : 초월
        Link  : 연결
        HyperLingk : 현재 문서를 넘어서 다른 문서와 연결

        Hyper Text : 클릭하면 연결된 다른 문서로 넘어간느 글자(문자열)

        HTML(Hyper Text Markup Language)
        : 다른 문서와 연결될 수 있는 글자를 작성할 수 있고 
          태그 형태를 해석해서 화면에 출력하는 언어

        HTTP(Hyper Text Transfer Protocol)
        : 인터넷을 Hyper Text가 작성된 문서(HTML)를
         서로 전송할때의 약속(규약)

a(Anchor, 연결) 태그

        - 현재 문서(HTML)와 다른 문서(HTML)를 연결 시켜
         클릭 시 이동하게 만드는 태그

        - a태그로 감싸진 글/내용/이미지 == Hyper Text

        - Hyper Text는 다른 문서와 구분되는 HTML 만의 핵심 기능!
         (이였다가 요즘은 다들 잘씀)

a 태그 사용법 1 : 페이지(문서) 이동

       [a 태그 속성]
      -  href(hyper text REFernce) : 연결할 주소를 작성하는 속성

      - target : a태그 클릭시 현재탭, 새 탭, 새 창 중 어떻게 열 것인지 지정
  1. 글자 관련 태그 이동
  2. 목력 관련 태그 이동
  3. 연습 문제 1 새 탭으로 열기

페이지 내에서 점프하기

        - 한 페이지 내에서 
         지정된 id 속성 값을 가진 요소로 이동

        * id(Identifier, 식별자(== 구분되는 글자)) 속성
        - 요소를 구분하는 용도의 속성
        - 하나의 html 파일 내에 
         같은 id 속성 값을 지닌 요소가 
          있어서는 안된다!
