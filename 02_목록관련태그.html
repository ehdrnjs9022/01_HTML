지동권의 페이지

ul (Unordered List) 태그 : 순서가 없는 목록

li (list item) 태그 : 목록의 한 줄

  • 맘스터치
    • 싸이버거
    • 어쩌고
  • 어쩌고
  • 저쩌고

ol(ordered list) : 순서가 있는 목록

  1. 우육면
  2. 김밥
  3. 쉑쉑버거
     * type 속성 : 표시되는 숫자 형식 지정
     type="a" : 영어 소문자 (a,b,c,d...)
     type="A" : 영어 대문자 (A,B,C,D...)
     type="i" : 로마 소문자 (i, ii, iii, ...)
     type="I" : 로마 대문자 (I, II, III, ...)
     type="1" : 숫자 (1,2,3, ...)    
* start 속성 : 목록 시작 번호
* reversed  : 순서 반대로 (역순)
  1. HTML5
  2. CSS3
  3. JavaScript
  4. Java
  5. DB(Oracle)

Emmet : HTML, CSS 빠른 자동완성 지원 플러그인 

    태그명 : 태그 시작 / 종료 태그 자동 완성

    > : 태그 내부에 작성 (하위생성)

    + : 태그 다음에 작성 (형제)

    *숫자 : 숫자 만큼 생성

    {} : 태그 내부에 작성될 내용 (중괄호, 블럭)

    $ : 1부터 1씩 증가하는 숫자 (시퀀스)
